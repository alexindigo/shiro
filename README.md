# Shiro

Shiro (シロ) – white, also part of shirofukurou (シロフクロウ) snowy owl,
satelite app for Fukuro – What? Where? When? game management system.

## Install

```
npm install shiro
```

## Run
```
node server.js --port 1337 --admin=secretpassword
```
